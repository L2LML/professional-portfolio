# Project 1 — IT Center Data Center & NOC Build-Out (Construction / Infrastructure)

Illustrative, self-directed portfolio project — not a record of an actual Authority engagement. See the [toolkit README](../README.md) for the full scenario and how this pairs with [Project 2 — Technology](../02-technology-project).

## Scenario

The IT Center's existing server room is converted office space with no redundant power or cooling path — a single point of failure for the systems that run baggage handling, passenger processing, and security. This project constructs a dedicated ~9,000 sq. ft. data center/NOC addition with N+1 electrical and mechanical redundancy, to be handed over to the companion Technology project for systems fit-out and cutover.

| | |
|---|---|
| **Budget** | $4,250,000 GMP (+ $340,000 / 8% contingency) |
| **Duration** | 14 months |
| **Delivery model** | Design-build GC with MEP Engineer of Record and independent Commissioning Agent |
| **Key milestone** | Substantial Completion & handover to IT Operations, Month 14 |

## PM disciplines shown

Capital-project chartering · GMP/capital budget tracking · GC and multi-trade subcontractor coordination · AHJ permitting and inspection management · quality management (code-compliance QA, inspection & test checkpoints, commissioning-based QC) · construction risk management (long-lead procurement, weather, unforeseen conditions) · four-level commissioning process · safety governance · facility handover to operations.
