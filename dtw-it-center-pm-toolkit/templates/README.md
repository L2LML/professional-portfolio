# PM Toolkit — Blank Templates

The reusable, fillable version of the three-document PM system used by both example projects in this toolkit. Use these as the starting point for any new project.

| File | Purpose |
|---|---|
| `PM_Toolkit_01_Charter_Comms_Training_TEMPLATE.docx` | Project Charter + Stakeholder Communication Management Plan + Training/Transition/Commissioning Plan |
| `PM_Toolkit_02_Workbook_TEMPLATE.xlsx` | WBS/Schedule, Budget Tracker, RACI Matrix, Stakeholder Register, Risk Register, Issue & Change Log, Vendor/Contractor Register |
| `PM_Toolkit_03_Status_Closeout_TEMPLATE.docx` | Recurring status report + final project closeout report |

**How to use:** replace every bracketed `[placeholder]`, and in the workbook, replace or delete the yellow-highlighted example row on each tab (see the workbook's `Instructions` tab for the full legend). Don't overtype the formula cells (Budget Tracker variance/%, Risk Register score) — they recalculate automatically.

See [`/01-construction-infrastructure-project`](../01-construction-infrastructure-project) and [`/02-technology-project`](../02-technology-project) for worked examples of this same template set fully filled in.
